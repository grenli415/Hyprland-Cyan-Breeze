 ИНСТРУКЦИЯ ПО УСТАНОВКЕ КОНФИГОВ (RUSSIAN / ENGLISH)
=========================================================

[РУССКИЙ]
Как применить эти конфигурационные файлы:

1. Hyprland:
   Скопируйте папку hyprland в ~/.config/
   Команда: cp -r ~/configs/hyprland ~/.config/

2. Waybar:
   Скопируйте папку waybar в ~/.config/
   Команда: cp -r ~/configs/waybar ~/.config/

3. Wofi:
   Скопируйте папку wofi в ~/.config/
   Команда: cp -r ~/configs/wofi ~/.config/

4. WezTerm:
   Скопируйте папку wezterm в ~/.config/
   Команда: cp -r ~/configs/wezterm ~/.config/

5. Neovim:
   Скопируйте папку nvim в ~/.config/
   Команда: cp -r ~/configs/nvim ~/.config/

6. Dolphin:
   Скопируйте файл dolphinrc в ~/.config/
   Команда: cp ~/configs/dolphin/dolphinrc ~/.config/

7. Control Center (Ваша программа):
   Скопируйте папку control-center в ~/.config/ (или туда, где программа ищет конфиг)
   Команда: cp -r ~/configs/control-center ~/.config/

8. Иконки:
   Скопируйте папку с иконками в ~/.local/share/icons/ или ~/.icons/
   Команда: cp -r ~/configs/icons ~/.local/share/icons/my-icons


[ENGLISH]
How to apply these configuration files:

1. Hyprland:
   Copy the hyprland folder to ~/.config/
   Command: cp -r ~/configs/hyprland ~/.config/

2. Waybar:
   Copy the waybar folder to ~/.config/
   Command: cp -r ~/configs/waybar ~/.config/

3. Wofi:
   Copy the wofi folder to ~/.config/
   Command: cp -r ~/configs/wofi ~/.config/

4. WezTerm:
   Copy the wezterm folder to ~/.config/
   Command: cp -r ~/configs/wezterm ~/.config/

5. Neovim:
   Copy the nvim folder to ~/.config/
   Command: cp -r ~/configs/nvim ~/.config/

6. Dolphin:
   Copy the dolphinrc file to ~/.config/
   Command: cp ~/configs/dolphin/dolphinrc ~/.config/

7. Control Center (Your program):
   Copy the control-center folder to ~/.config/ (or wherever your program expects it)
   Command: cp -r ~/configs/control-center ~/.config/

8. Icons:
   Copy the icons folder to ~/.local/share/icons/ or ~/.icons/
   Command: cp -r ~/configs/icons ~/.local/share/icons/my-icons
