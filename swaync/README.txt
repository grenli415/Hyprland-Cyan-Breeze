# SwayNC Configuration Information (RUS/ENG)

[РУССКИЙ]
Настройки SwayNC (шторка уведомлений) хранятся в файле config.json.
Поскольку формат JSON не поддерживает комментарии, мы оставили файл чистым.
Он отвечает за позицию шторки, отступы, ширину окна и анимации.

[ENGLISH]
SwayNC (notification center) settings are stored in config.json.
Since JSON format does not support comments, we left the file clean.
It controls the position, margins, window width, and animations of the notification center.
